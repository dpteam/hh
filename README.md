Hacker-History: A text game

Installing:
* 1. Install the [QSP Engine](http://qsp.su "QSP Engine")
* 2. Open the game with it.

Edit the source:
* 1. Install the [qGen Editor](http://vk.cc/2kyC1U "qGen Editor")
* 2. Open the game with it.
